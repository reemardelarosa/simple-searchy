### General Info `<simple-search>`

`<simple-search>` shows a search input area on your page.

To use, insert `<simple-search>` in the body:
```html
    <body>
      <simple-search></simple-search>
```
Example:

To show the search 
```html
    <body>
      <simple-search static-open></simple-search>
```
